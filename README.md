# Node.js Workflows

This repository contains a number of example workflows for Node.js projects.

- [Scan for license violations](https://github.com/actions/npm-workflows/tree/master/workflows/license-scan-pull-request)—Scan pull requests for approved license violations using [licensed](https://github.com/github/licensed).
- [Test with multiple Node.js versions](https://github.com/actions/npm-workflows/tree/master/workflows/test-multiple-node-versions)—Tests your code with multiple configurable versions of Node.js.
- [Label pull requests automatically](https://github.com/actions/npm-workflows/tree/master/workflows/triage-label-pull-requests)—Label pull requests automatically based on a configuration yaml file.
- [Vulnerability scan](https://github.com/actions/npm-workflows/tree/master/workflows/vulnerability-scan-pull-request)—Scan your project's dependencies for known vulnerabilities.
- [Welcome first-time contributors](https://github.com/actions/npm-workflows/tree/master/workflows/welcome-first-time-contributor)—Welcome first-time contributors to your project by automatically posting a welcome message on their pull request.
